# Chapter 4: Hardware Requirements for Physical AI

## Overview
This course is computationally demanding: Physics Simulation, Visual Perception, and Generative AI all require powerful hardware.

### 1. Digital Twin Workstation (Required per Student)
- GPU: NVIDIA RTX 4070 Ti or higher  
- CPU: Intel Core i7 13th Gen+ / AMD Ryzen 9  
- RAM: 64 GB DDR5 (32 GB minimum)  
- OS: Ubuntu 22.04 LTS

### 2. Physical AI Edge Kit
- Brain: NVIDIA Jetson Orin Nano (8GB) or Orin NX (16GB)  
- Eyes: Intel RealSense D435i/D455  
- Inner Ear: USB IMU BNO055  
- Voice: USB Microphone/Speaker (ReSpeaker)

### 3. Robot Lab Options
- **Proxy Approach:** Quadruped or robotic arm  
- **Miniature Humanoid:** Small table-top humanoids  
- **Premium Lab:** Full humanoid like Unitree G1  

### 4. Cloud-Based Lab (High OpEx)
- Cloud Workstations (AWS/Azure, NVIDIA Isaac Sim)  
- Local "Bridge" Edge Devices (Jetson kits)

---

This chapter ensures students understand the hardware and cloud requirements for the course.
